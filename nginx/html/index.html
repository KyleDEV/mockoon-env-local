Mockoon Server - Mocking API

NginX가 /api/ 하위에 Mockoon 엔드포인트를 연결했습니다.

Mockoon API evironment 데이터 교체 방법

/mockoon/data/evergreen.json을 원하는 데이터로 교체한 뒤 /moockoon-env-local 에서 docker-compose의 mockoon-cli를 재시작해주세요

  • Mockoon 서비스만 재시작하려면:
    • docker-compose restart mockoon
  • 이미지를 다시 빌드하고 재시작하려면:
    • docker-compose build mockoon
    • docker-compose up -d mockoon
