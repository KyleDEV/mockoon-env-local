Mockoon Server

NginX가 /api/ 하위에 Mockoon 엔드포인트를 연결했습니다.

  • Mocking API: http://localhost:8080/api/{엔드포인트}
  • Swagger UI: http://localhost:8080/swagger/
  • Mockoon 공식사이트, GitHub

Mockoon API Environment 데이터 교체 방법

/mockoon/data/evergreen.json을 원하는 데이터로 교체한 뒤 /moockoon-env-local 에서 docker-compose의 mockoon을 재시작해주세요. mockoon을 재시작하면 Swagger UI 또한 업데이트 됩니다.

Mockoon 서비스 관리
서비스 재시작

Mockoon 서비스만 재시작하려면:

  • docker-compose restart mockoon
이미지를 다시 빌드하고 재시작

이미지를 다시 빌드하고 재시작하려면:

  • docker-compose build mockoon
  • docker-compose up -d mockoon
Docker Compose 전체 관리
전체 서비스 재시작

전체 서비스를 재시작하려면:

  • docker-compose restart
전체 이미지 빌드 및 재시작

전체 이미지를 다시 빌드하고 재시작하려면:

  • docker-compose build
  • docker-compose up -d
전체 서비스 중지

전체 서비스를 중지하려면:

  • docker-compose down
